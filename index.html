Project Overview

This project involves making a set of programs that will allow us to type coordinates and have the board display a compass that points from our current location to the target location.

Problem

Maps are slow and overcomplicated, need a compass that points directly to where we want to go.

Proposed Solution

We introduce a simple system that points directly from point A to point B.
